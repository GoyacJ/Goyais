# Goyais Monorepo

This repository is organized by platform modules.

## Modules

- `go_server`: Go backend service
- `vue_web`: Vue web frontend
- `java_server`: Java backend placeholder
- `python_server`: Python backend placeholder
- `flutter_mobile`: Flutter mobile placeholder
- `android_mobile`: Android mobile placeholder

## Docs

- Root business/governance docs: `docs/`
- Go technical docs: `go_server/docs/`
- Web technical docs: `vue_web/docs/`

## Codex Governance

- Root policy: `AGENTS.md`
- Compatibility entry: `.agents/AGENTS.md`
- Stable rules: `.agents/rules/`
- Reusable skills: `.agents/skills/`
- Source header policy (SPDX/Author/Created/Version/Description): `go_server/scripts/ci/source_header_check.sh`

## Command Entry

Run commands inside each module directory.
