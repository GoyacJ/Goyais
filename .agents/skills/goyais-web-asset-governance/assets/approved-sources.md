# Approved Sources (Asset Whitelist)

## Allowed by default

- Heroicons
  - Site: https://heroicons.com/
  - Repo: https://github.com/tailwindlabs/heroicons
  - License: MIT
- Lucide (optional alternative)
  - Repo: https://github.com/lucide-icons/lucide
  - License: ISC
- Apache-2.0 / CC0 / MIT / ISC resources with explicit license URLs

## Conditionally allowed (explicit terms)

- unDraw
  - Site: https://undraw.co/illustrations
  - License page: https://undraw.co/license
  - Use only if terms are documented in notices and runtime does not hotlink.

## Blocked

- Unknown license or no license link.
- Third-party CDN hotlink at runtime.
- Bulk pack redistribution or prohibited derivative usage against source terms.
