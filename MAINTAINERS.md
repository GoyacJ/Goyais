# Maintainers

[简体中文](MAINTAINERS.zh-CN.md) | English

This file tracks governance contacts and responsibilities.

## Maintainer Groups

- Core Maintainers: repository technical direction and release approval
- Security Maintainers: vulnerability triage and disclosure handling
- Release Maintainers: release notes, tagging, and rollout checklist

## Operating Rule

Before first public release, this file should be updated with explicit GitHub handles or teams.
