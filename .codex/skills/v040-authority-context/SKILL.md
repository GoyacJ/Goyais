# v0.4.0 权威上下文技能

用于处理产品定义、架构边界、范围判定、验收口径相关问题。

## 触发条件

- 用户询问行为定义、边界、范围、架构期望。
- 方案或计划引用来源不明确或疑似越权。

## 输入

- 当前问题或任务
- v0.4.0 文档集：
  - `docs/v_0_4_0/PRD.md`
  - `docs/v_0_4_0/TECH_ARCH.md`
  - `docs/v_0_4_0/IMPLEMENTATION_PLAN.md`
  - `docs/v_0_4_0/DEVELOPMENT_STANDARDS.md`

## 工作流

1. 识别问题类型：
   - 业务语义
   - 架构不变量
   - 执行/阶段顺序
   - 工程门禁/DoD
2. 只提取 v0.4.0 的相关章节。
3. 按 PRD 优先原则解决冲突。
4. 返回简明约束与边界结论。

## 输出契约

- 问题的简明答案。
- 采用的权威约束清单。
- 若需改动，给出受影响文档列表。

## 护栏

- 不得将 v0.1/v0.2/v0.3 文档作为实现依据。
- 必须使用 `Conversation` 作为主术语。
- 必须保持 Hub 权威架构语义。
- 与用户沟通默认使用中文。
